PNLE Review Hub - Static Website
--------------------------------
Files included:
- index.html : Home / Dashboard (Study plan, quick links)
- study.html : Topic summaries, flashcards, practice quiz (client-side)
- resources.html : Links & PDF resources (placeholders)
- about.html : About and how to use
- data.js : Questions, flashcards and topic data used by study.html
- manifest.json, icons/ : PWA support
- assets/style.css : site styles

Quick hosting (GitHub Pages):
1. Create a new GitHub repository.
2. Upload all files to the repository root (or push from local git).
3. In Settings -> Pages, set Source to 'main' branch and root (/).
4. After a minute you'll have https://<your-username>.github.io/<repo>/

Or use Netlify Drop: https://app.netlify.com/drop
